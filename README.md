# 🎭 마피아 파티 (진행자 없이 즐기는 실시간 마피아)

친구들과 여행 가서 휴대폰만으로 즐기는 마피아 게임입니다.
**진행자(사회자)가 필요 없습니다.** 서버가 역할 배정, 밤/낮 전환, 능력 처리, 개표, 승패 판정을 모두 자동으로 진행합니다.

- 👥 **8 ~ 12인**
- 📱 각자 **자기 휴대폰**으로 접속 (밤에 마피아·연인끼리만 보이는 비밀 채팅)
- 🕵️ **특수 능력 시민** 다수 (경찰, 의사, 군인, 영매, 정치인, 기자, 마담, 연인 등)

---

## 🎮 게임 방법

1. 한 명이 **새 방 만들기** → 화면에 뜨는 **4자리 방 코드**를 친구들에게 알려줍니다.
2. 나머지는 닉네임 + 방 코드로 **참가**합니다.
3. 8명 이상 모이면 **방장(👑)** 이 [게임 시작]을 누릅니다.
4. 각자 자기 직업을 확인하고 게임이 진행됩니다.
   - 🌙 **밤**: 능력자는 대상을 지목하고, **마피아끼리 / 연인끼리** 비밀 채팅을 합니다. (다른 사람에게는 안 보입니다)
   - ☀️ **낮**: 모두 함께 토론하고, 마피아로 의심되는 사람을 **투표**로 처형합니다.
5. 마피아를 전부 잡으면 **시민 승리**, 마피아 수가 시민 이상이 되면 **마피아 승리**.

> 시간이 남거나 빨리 넘기고 싶으면 방장이 **[⏭️ 단계 넘기기]** 로 진행할 수 있습니다.

### 👁️ 혼자 구경하기 (봇 데모 관전)

처음 화면 맨 아래 **[👁️ 봇 8명 데모 관전]** 을 누르면, AI 봇 8명이 자동으로 한 판을 플레이하는 과정을 **신(神) 시점**으로 지켜볼 수 있습니다.
- 모든 플레이어의 **직업이 공개**되어 보입니다.
- 밤의 **마피아 비밀 채팅·연인 비밀 채팅**까지 관전자에게는 전부 보입니다 (전체/마피아/연인 탭).
- 룰과 흐름을 익히거나, 게임이 어떻게 굴러가는지 미리 확인할 때 유용합니다.

## 🧑‍🤝‍🧑 직업 안내

### 🔴 마피아 진영
| 직업 | 능력 |
|---|---|
| 🔪 **마피아** | 밤마다 동료와 비밀 채팅하며 시민 1명을 제거 |
| 🕵️ **스파이** | 마피아 편이지만 경찰 조사에 **"시민"으로 위장**됨 (10인+) |

### 🔵 시민 진영
| 직업 | 능력 |
|---|---|
| 👮 **경찰** | 밤마다 1명을 조사해 마피아 여부 확인 |
| 💉 **의사** | 밤마다 1명을 보호 (마피아 공격 무효, 자신도 가능) |
| 🪖 **군인** | 마피아의 첫 공격을 1회 버팀 |
| 🔮 **영매** | 전날 죽은 사람의 진영을 알게 됨 |
| 🎩 **정치인** | 낮 투표에서 **2표** 행사 |
| 📰 **기자** | 밤에 1명 취재 → 다음 날 그 직업이 전체 공개 (1회) |
| 💋 **마담** | 밤에 1명 유혹 → 다음 날 능력·투표 봉쇄 |
| 💞 **연인** | 둘이 밤에 비밀 채팅, 한 명이 죽으면 다른 한 명도 사망 |

인원수에 따라 직업 구성이 자동으로 조정됩니다 (로비에서 미리 확인 가능).

## 💻 로컬에서 바로 실행 (같은 와이파이)

```bash
npm install
npm start
```

서버를 띄운 컴퓨터의 IP를 확인해 친구들이 같은 와이파이에서 접속합니다.

- 호스트 컴퓨터: `http://localhost:3000`
- 같은 와이파이의 다른 폰: `http://<호스트IP>:3000` (예: `http://192.168.0.5:3000`)
- IP 확인: Windows `ipconfig` → IPv4 주소

## ☁️ 무료 클라우드 배포 (어디서든 접속)

여행지에서 와이파이가 제각각이어도 되도록 **인터넷 주소 하나로** 접속하게 만듭니다.

### 방법 A — Render (추천, 무료)

1. 이 폴더를 GitHub 저장소에 올립니다.
   ```bash
   git init
   git add .
   git commit -m "마피아 게임"
   git branch -M main
   git remote add origin https://github.com/<본인계정>/mafia-party.git
   git push -u origin main
   ```
2. [render.com](https://render.com) 가입 후 **New + → Blueprint** 선택 → 방금 올린 저장소 연결.
   (저장소에 포함된 `render.yaml` 이 설정을 자동 적용합니다.)
   - 또는 **New + → Web Service** 로 직접 설정:
     - Build Command: `npm install`
     - Start Command: `npm start`
3. 배포가 끝나면 `https://mafia-party-xxxx.onrender.com` 같은 주소가 나옵니다. 이 링크를 친구들에게 공유하세요.

> ⚠️ Render 무료 플랜은 15분간 접속이 없으면 잠들어, 첫 접속 시 30초~1분 깨어나는 시간이 걸릴 수 있습니다. 게임 시작 전 한 번 미리 열어두면 됩니다.

### 방법 B — Railway

1. 위와 같이 GitHub에 올립니다.
2. [railway.app](https://railway.app) → **New Project → Deploy from GitHub repo** 선택.
3. 자동으로 `npm install` → `npm start` 가 실행됩니다. 별도 설정이 거의 필요 없습니다.
4. **Settings → Networking → Generate Domain** 으로 공개 주소를 만들어 공유합니다.

배포 환경에서는 포트가 `PORT` 환경변수로 주입되며, 이 서버는 이미 `process.env.PORT` 를 사용하도록 만들어져 있어 추가 설정이 필요 없습니다.

## 🛠️ 기술 구성
- **백엔드**: Node.js + Express + Socket.IO (실시간 양방향 통신)
- **프론트**: 순수 HTML/CSS/JS (모바일 최적화, 별도 빌드 불필요)
- **상태 저장**: 메모리 기반 (DB 불필요 — 파티용)
- 새로고침/폰 잠금 후에도 **자동 재접속**으로 게임 복귀

## 파일 구조
```
server.js   - Express + Socket.IO 서버, 접속/이벤트 라우팅
game.js     - 게임 진행 로직 (밤/낮 상태머신, 능력 처리, 승패)
roles.js    - 직업 정의 및 인원수별 배정표
public/     - 클라이언트 (index.html, style.css, client.js)
```
